## 📁 폴더 구조
```
Synamon
├─ public
│  └─ images                # 이미지, 아이콘 리소스
├─ src
│  ├─ apis                  # API 사용을 위한 세팅
│  ├─ components            # 공용 컴포넌트
│  ├─ constants             # 재사용 모음
│  ├─ hooks                 # 커스텀 훅
│  ├─ lib                   # axios instance 모음
│  ├─ pages                 # 페이지 모음
│  ├─ styles                # 스타일 전역 파일
│  ├─ types                 # 주요 type
└─ └─ utils                 # 기타 유틸리티
```
